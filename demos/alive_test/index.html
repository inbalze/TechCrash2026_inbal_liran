Alive Test

Bidirectional UART between DE10-Lite FPGA and ESP32 — verifies the full communication link is alive.

What It Does

A minimal system test that proves both directions of the FPGA-to-ESP32 link work:

  • FPGA → ESP32: The FPGA counts 0→9 on HEX0 (one digit per second) and transmits each digit as ASCII over UART.
  • ESP32 → FPGA: The ESP32 counts 9→0 (countdown) and transmits each digit to the FPGA, which displays it on HEX1.
  • The ESP32 also shows both values on its OLED display, along with button states and potentiometer reading.
  • LEDs sweep left-to-right as a visual heartbeat indicator.

Build & Flash

FPGA

cd demos/alive_test/fpga quartus_sh --flow compile alive_test quartus_pgm -m jtag -o "p;output_files/alive_test.sof"

ESP32

cd demos/alive_test/esp32 pio run -t upload

Expected Behavior

  • HEX0 (rightmost): Counts 0 → 9 → 0 → ... (one step per second)
  • HEX1 (second from right): Shows the ESP32 countdown (9 → 8 → ... → 0 → 9)
  • HEX2-5: Blank (all segments off)
  • LEDs: Sweep left-to-right
  • ESP32 OLED: Shows "ALIVE TEST", button states, potentiometer bar, and RX:<digit> TX:<digit>
